                  <p>
                    O cartão de débito é uma forma de pagamento eletrônica que permite a dedução do valor de uma compra diretamente na conta bancária do titular do cartão (corrente ou poupança).
                  </p>
                </div>
              </li>

              <li>
                <div data-bs-toggle="collapse" href="#faq2" class="collapsed question">O que é o IBAN? <i class="bi bi-chevron-down icon-show"></i><i class="bi bi-chevron-up icon-close"></i></div>
                <div id="faq2" class="collapse" data-bs-parent=".faq-list">
                  <p>
                    O IBAN é um código-padrão internacional para a identificação de contas bancárias. O IBAN é composto, no máximo, por 34 caracteres. Os dois primeiros caracteres correspondem ao país de domiciliação da conta (PT-Portugal, ES-Espanha, DE-Alemanha, etc.). O terceiro e quarto caracteres são de controlo e servem para validação do código do país. Os restantes dígitos correspondem à estrutura de identificação de contas definida para cada país.
                  </p>
                </div>
              </li>

              <li>
                <div data-bs-toggle="collapse" href="#faq3" class="collapsed question">Poderei fazer pagamentos para fora de Portugal? <i class="bi bi-chevron-down icon-show"></i><i class="bi bi-chevron-up icon-close"></i></div>

                    Sim, caso possua o plano <b>plus</b> ou <b>premium</b>. Desde outubro de 2020 é possível realizar transferências imediatas entre países do espaço SEPA. Porém este serviço é facultativo, pelo que deverá verificar quais são os prestadores de serviços de pagamento aderentes.                  </p>
                </div>
              </li>

              <li>
                <div data-bs-toggle="collapse" href="#faq6" class="collapsed question">Como é a minha palavra passe protegida no plano premium? <i class="bi bi-chevron-down icon-show"></i><i class="bi bi-chevron-up icon-close"></i></div>
